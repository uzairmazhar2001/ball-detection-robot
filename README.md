# Ball Detection Using Vision in Robot Soccer Simulation

## Project Description
This project implements a robot that detects a soccer ball using a camera in Webots. The robot processes RGB image data to identify an orange ball and move towards it.

## Features
- Camera-based ball detection
- RGB color filtering
- Robot movement towards ball
- Ball pushing behavior

## Tools Used
- Webots simulator
- Python

## How to Run
1. Open Webots
2. Load the .wbt world file
3. Run the simulation
